## Dev Environment  
in order to test the app you the following tools must be installed in your OS:<br>
* ruby 
* rails
* mongodb
* nodejs
## Install App Dependencies
```
bundle install
cd angular-front-end
npm i
```

## start the App
start backend server
```
rails s
```
frontend server
```
cd angular-front-end
npm run start
```
start app in web browser at:<br>
http://localhost:4200/
