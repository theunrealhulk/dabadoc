DabaDoc Coding challenge

this a simple web application with ruby on rails on the backend and angular on the frontend
the purpose of the app is general topic questions related to a location kind of app where the user asks a question about a specific location.

Dev Environment

in order to test the app you the following tools must be installed in your OS:
* ruby * rails * mongodb * nodejs

Install App Dependencies

bundle install
cd angular-front-end
npm i

start the App

start backend server

rails s

frontend server

cd angular-front-end
npm run start

start app in web browser at:
localhost:4200/
